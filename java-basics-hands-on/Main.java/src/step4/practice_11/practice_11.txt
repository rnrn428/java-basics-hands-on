 배열 (Array) vs ArrayList

 배열 (Array) -> String[] arr = new String[3]; // 공간 크기 먼저 설정
                arr[0] = "값"; // 첫 번째 인덱스에 "값"이라는 값 저장.
                arr[0] // 첫 번째 저장했던 값
                arr.length // 공간 크기


ArrayList -> ArrayList<String> list = new ArrayList<> ();
                list.add("값"); // 순서대로 값 넣기
                list.get(0); // 리스트 첫 번째 값 가져오기
                list.size(); // 넣어져 있는 공간 크기 구하기
